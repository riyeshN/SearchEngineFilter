# DJANGO:

1) Clone the project:
2) Create venv:
  * Windows Commandline:
    python -m venv venv
    venv\Scripts\activate
3) pip install -r requirements.txt
4) python manage.py makemigrations
5) python manage.py migrate
6) python manage.py runserver

# REACT:
1) npm install
2) npm run dev
